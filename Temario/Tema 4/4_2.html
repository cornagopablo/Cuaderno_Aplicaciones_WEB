Instalación y configuración de moodle

Existen dos maneras de hacer la instalación de moodle:

  1. En equipos que tienen instalado XAMPP.
    1. Cogemos el archivo moodle-latest-26.zip y lo descomprimimos en la carpeta htdocs (XAMPP). No hay que crear una carpeta a parte, al descomprimir lo hace sobre una carpeta de nombre moodle.
      • C:\xampp\htdocs
    2. Abrimos XAMPP y ejecutamos los servicios de Apache y MySQL.
    3. Abrimos un navegador web e ingresamos la dirección:
      • localhost
    4. Accedemos a phpMyAdmin.
    5. Pulsamos sobre Usuarios y seguido pulsamos en Agregar Usuario.
      • Nombre de usuario:
        • moodle
      • Servidor:
        • localhost
      • Contraseña:
        • moodle
      • Generar contraseña:
        • moodle

        • Marcamos la casilla:

        • Crear base de datos con el mismo nombre y otorgar todos los privilegios

        En Privilegios globales marcamos la opción de Marcar todos.

    6. Le damos a continuar y cerramos la ventana.

    1. Editamos el archivo php.ini en C:\xampp\php\php.ini.
      • max_execution_time=30 y lo cambiamos por max_execution_time=600.
    2. En el navegador tecleamos ahora la dirección:
      • localhost/moodle
    3. Elegimos como idioma:
      • Español - Internacional (es)
    4. En las rutas no tocamos nada, lo dejamos tal cual está.
    5. En base de datos seleccionamos:
      • MySQL mejorado (native/mysqli)
    6. Ajustes de base de datos
      • Servidor de la base de datos:
        • localhost
      • Nombre de la base de datos:
        • moodle
      • Usuario de la base de datos:
        • root
      • Contraseña de la base de datos:
        • *Lo dejamos en blanco*
      • Puerto de la base de datos:
        • 3306 *Dependiendo del puerto que tengamos abierto para MySQL*
    7. Le damos a continuar. Si en la siguiente pantalla nos sale "Su entorno de servidor cumple todos los requerimientos mínimos" en verde, le damos a continuar.

    1. Esperamos a que cargue todo y le damos a continuar

    1. Instalación/General
      • Nombre de usuario:
        • admin
      • Nueva contraseña:
        • Vespino2021*
      • Dirección de correo:
        • *Nos inventamos la dirección*
      • El resto de datos los dejamos tal cual están
      • Le damos a actualizar información personal

    1. Ajustes de la página principal.
      • Nombre completo del sitio:
        • PRIMEROS PASOS MOODLE
      • Nombre corto para el sitio:
        • mymoodle
      • Descripción de la página principal:
        • Aprendiendo a gestionar un sistema de aprendizaje online.



  2. Sin necesidad de tener instalado XAMPP. La instalación se realiza mediante un paquete todo incluido que integra los servicios de Apache y MySQL.
    1. Paramos los servicios abiertos de XAMPP y cerramos el programa.
    2. Cogemos el archivo MoodleWindowsInstaller-latest-26.zip y lo descomprimimos en la raíz del disco duro, en una carpeta de nombre modle.
    3. Editamos el archivo php.ini en C:\moodle\server\php\php.ini.
      • max_execution_time=30 y lo cambiamos por max_execution_time=600.
    4. Ejecutamos Start Moodle.exe. No cerramos la ventana del cmd que nos abre.
    5. Si nos pide permisos, se los damos.
    6. Abrimos un navegador web y tecleamos la dirección:
      • localhost
    7. Elegimos como idioma:
      • Español - Internacional (es)
    8. En las rutas no tocamos nada, lo dejamos tal cual está.
    9. En base de datos seleccionamos:
      • MySQL mejorado (native/mysqli)
    10. Ajustes de base de datos
      • Servidor de la base de datos:
        • localhost
      • Nombre de la base de datos:
        • moodle
      • Usuario de la base de datos:
        • root
      • Contraseña de la base de datos:
        • *Lo dejamos en blanco*
      • Puerto de la base de datos:
        • 3306 *Dependiendo del puerto que tengamos abierto para MySQL*
    11. Le damos a continuar. Si en la siguiente pantalla nos sale "Su entorno de servidor cumple todos los requerimientos mínimos" en verde, le damos a continuar.

        • admin
      • Nueva contraseña:
        • Vespino2021*
      • Dirección de correo:
        • *Nos inventamos la dirección*
      • El resto de datos los dejamos tal cual están
      • Le damos a actualizar información personal

    1. Ajustes de la página principal.
      • Nombre completo del sitio:
        • PRIMEROS PASOS MOODLE
      • Nombre corto para el sitio:
        • mymoodle
      • Descripción de la página principal:
        • Aprendiendo a gestionar un sistema de aprendizaje online.
