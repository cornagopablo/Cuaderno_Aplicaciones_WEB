Práctica uso plugin 1

  1. Crear dos artículos dentro de Aficiones:

    • El primero, permite visualizar 2 videos
    • El segundo, permite escuchar 2 canciones
  2. Añadir al menú de Otras Cosas, dos items nuevos:
    • Mis Videos
    • Mi Música

  1. Crear un artículo:
    • Título:
      • ¿Dónde estoy?
    • Categoría:
      • Otras Cosas
    • Tiene que incluir un mapa de Google Maps indicando la ubicación del IES Virgen del Espino
  2. Crear un elemento de menú en Otras Cosas para visualizar el artículo ¿Dónde estoy?

    • De lunes a domingo

  1. Añadir dos componentes más:
    • Reloj
    • Contador de las visitas de hoy

Práctica uso Plugin 6

  1. Añadimos un componente nuevo y su traducción, que sirve para implementar la opción de descargar archivos a un artículo.

  1. Dentro del artículo de Sistemas Operativos Monousuario, incluimos para descargar el archivo:
    • HistoriaInformatica.pdf

  1. Creamos un artículo llamado Descargas, que esté dentro de Aplicaciones Web. Debe permitir descargar 3 archivos.
    • Estilos_CSS.pdf
    • Manual_css1.pdf
    • EjemplosEstilos.doc

  1. Además, tendremos que crear un elemento de menú dentro de Apuntes, que abra ese artículo.


Práctica uso Plugin 7

  1. instalar un nuevo componente, que sirve para instalar una galería de vídeos.

  1. Los vídeos pueden estar almacenados en la plataforma, o ser visualizados online.
  2. Crear una categoría de vídeos dentro del menú Otras Cosas y crear un elemento de menú llamado Vídeos de Joomla.
    • Joomla
      • Un par de vídeos didácticos.

  1. Crear dos categorías.
    • Una para Redes, con dos vídeos.
    • Una para Ofimática, con dos vídeos.

  1. Crear un nuevo menú, llamado Mis Vídeos, con tres elementos.
    • Uno para visualizar vídeos de la categoría Ofimática.
    • Uno para visualizar vídeos destacados (1 de cada categoría).
    • Uno para visualizar todos los vídeos.
